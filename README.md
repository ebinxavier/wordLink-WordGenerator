# WordLink Word Generator Solution.
* Clone the repo.
* Install dependencies using `npm i`.
* Run using `npm start`
  - Command   : `npm start [WORD] [COUNT](Optional)`
  
  - Examples  : `npm start angle`
  
  ```
          Set { 
          'angle', 
          'angel', 
          'glean', 
          'genal' 
          }
  ```
  
  - `npm start angle 4`
  
  
  ```
          Set {
          'agen',
          'neal',
          'gane',
          'gale',
          'gaen',
          'glen',
          'gean',
          'geal',
          'gena',
          'lang',
          'lane',
          'lean',
          'leng',
          'egal',
          'elan' 
          }
  ```
